另外一些特殊字符还有其他表示，也就是常说的 html 转义符：

空格 &nbsp; 也可以使用 &#160;

css 中使用 \ + 16进制的unicode编码

.unicode-css::before {
  content: '\70ed\0488\7684\0488\5b57\0488\90fd\0488\51fa\0488\6c57\0488\4e86\0488';
}

javascript 中使用 \u + 16进制的unicode编码

document.getElementById('#unicode-javascript').innerHTML = '\u70ed\u0488\u7684\u0488\u5b57\u0488\u90fd\u0488\u51fa\u0488\u6c57\u0488\u4e86\u0488';

另外 javascript 还有使用 0x 表示16进制。例如：

String.fromCodePoint('0x20bb7')
